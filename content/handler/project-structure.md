### How to Create the Project Structure

If your folder doesnâ€™t look like the minimal structure yet, here are the exact steps to create it from scratch.

#### 1. Create a new folder
```sh
mkdir cors-proxy
cd cors-proxy
```

#### 2. Initialize a Go module
```sh
go mod init cors-proxy
```

#### 3. Create your handler file
```sh
touch main.go
```
You will paste the Minimal Example handler into this file later.

#### 4. Create the build script
```sh
touch build.sh
chmod +x build.sh
```

#### 5. Add the AWS Lambda dependency
```sh

```
